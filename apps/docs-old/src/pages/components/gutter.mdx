# Gutter Component

This gutter control will align with parent element, it will update directly to parent element size.
To implement it, just place it into the element that you want to resize.

<Callout>
  Check out our [ladle](/ladle?story=gutter--default) for more info on this
  component.
</Callout>

---

### Import:

```js
import {Gutter} from '@comfortdelgro/react-compass'
```

or

```js
import Gutter from '@comfortdelgro/react-compass/gutter'
```

---

### Basic Gutter:

<section className={styles.gutterShowcaseContainer}>
  <GutterDefault />
</section>

```js
const ref = useRef < HTMLDivElement > null
const [width, setWidth] = useState < CSSProperties['width'] > undefined

return (
  <div style={{display: 'flex', height: '100%', flex: 1}}>
    <div
      ref={ref}
      style={{
        position: 'relative',
        width: width,
      }}
    >
      <div style={{overflow: 'hidden'}}>Right side</div>
      <Gutter parentRef={ref} onChange={setWidth} />
```

### Props

| Name            | Type                             | Default   | Description                                                                   |
| :-------------- | :------------------------------- | :-------- | :---------------------------------------------------------------------------- |
| parentRef\*     | `React.ReactNode`                | —         | Reference to the parent element                                               |
| css             | `CSS`                            | —         |                                                                               |
| side            | `'left'` \| `'right'`            | `'right'` | The side that you want to resize of it's parent.                              |
| hasExpandButton | `boolean`                        | false     | To display quick collapse icon button on top of gutter.                       |
| minCollapse     | `CSS['width'] `                  |           | The minimum width when gutter collapse.                                       |
| maxExpand       | `CSS['width'] `                  |           | The maximum width when gutter expand.                                         |
| hasExpandButton | `boolean`                        | false     | To display quick collapse icon button on top of gutter.                       |
| onChange        | `(width: CSS['width'] ) => void` | —         | Callback event that will return the width for the parent as the gutter resize |

# Dashboard Sidecard

Dashboard Sidecard presents section of data on dashboard.

<Callout>
  Check out our [ladle](/ladle?story=dashboard-sidecard--default) for more info
  on this component.
</Callout>

---

### Import:

```js
import {DashboardSidecard} from '@comfortdelgro/react-compass'
```

or

```js
import DashboardSidecard from '@comfortdelgro/react-compass/dashboard-sidecard'
```

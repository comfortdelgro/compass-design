# Search Fields

Search Fields let users enter search input.

<Callout>
  Check out our [ladle](/ladle?story=searchfield--variants) for more info on
  this component.
</Callout>

---

### Import:

```js
import {SearchField} from '@comfortdelgro/react-compass'
```

or

```js
import SearchField from '@comfortdelgro/react-compass/searchfield'
```

---

### Basic Search Field:

<section className={styles.avatarShowcaseContainer}>
  <SearchField placeholder='Search' onSubmit={(text) => console.log(text)} />
  <SearchField
    isDisabled
    placeholder='Search'
    onSubmit={(text) => console.log(text)}
  />
</section>

```js
<SearchField placeholder='Search' onSubmit={(text) => console.log(text)} />
<SearchField
  isDisabled
  placeholder='Search'
  onSubmit={(text) => console.log(text)}
/>
```

---

### Custom styling:

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

<section className={styles.avatarShowcaseContainer}>
  <SearchField
    placeholder='Search'
    onSubmit={(text) => console.log(text)}
    css={{border: '1px solid gray'}}
  />
</section>

```js
<SearchField
  placeholder='Search'
  onSubmit={(text) => console.log(text)}
  css={{border: '1px solid gray'}}
/>
```

---

### Props:

| Name       | Type               | Default | Description |
| :--------- | :----------------- | :------ | :---------- |
| isDisabled | `false`\| `true`   |         |             |
| onSubmit   | `(i:string)=>void` | —       |             |
| css        | `CSS`              | —       |             |

# Chart

The Chart using for visualize your data.

---

### Import:

```js

```

or

```js

```

### Props:

| Name    | Type      | Default | Description                |
|:--------|:----------|:--------|:---------------------------|
| dataSet | `DataSet` | -       | Data set of the chart      |
| unit    | `string`  | -       | Unit of the dataset lables |
| title   | `string`  | -       | Display title of the chart |
| css     | `CSS`     | —       |                            |

Type of DataSet includes:
| Name     | Type                               | Default | Description                                                                              |
| :------- | :--------------------------------- | :------ | :--------------------------------------------------------------------------------------- |
| labels   | `number[]`                         | -       | List of labels                                                                           |
| legends  | `string[]`                         | -       | List of kind of data                                                                     |
| data     | `{ title: string, data: number[]}` | -       | -                                                                                        |
|          | `title: string`                    | -       | Title of the point data                                                                  |
|          | `data: number[]`                   | -       | List of value in one point                                                               |
| css      | `CSS`                              | —       | The system prop that allows defining system overrides as well as additional CSS styles.  |

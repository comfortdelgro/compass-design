### Custom styling:

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

<section className={styles.avatarShowcaseContainer}>
  <Avatar
    label='Albert Einstein'
    image='https://i.pravatar.cc/150?u=a0425e81423429026704d'
    size='lg'
    css={{opacity: '0.8', '&:hover': {opacity: '0.5', cursor: 'pointer'}}}
  />
</section>

```js
<Avatar
  label='Albert Einstein'
  image='https://i.pravatar.cc/150?u=a0425e81423429026704d'

  css={{opacity: '0.8', '&:hover': {opacity: '0.5', cursor: 'pointer'}}}
```

### Props:

#### `<Avatar/>` :

| Name  | Type                         | Default | Description                                                                                          |
| :---- | :--------------------------- | :------ | :--------------------------------------------------------------------------------------------------- |
| label | `String`                     | —       | The label, used as fallback for image. The avatars would show the initials of the labels.            |
| image | `String `                    | —       | Image of the avatars takes the highest priority. We pass the image's address as string to this prop. |
| size  | `xs`\|`sm`\|`md`\|`lg`       | —       | Sizes of the avatars.                                                                                |
| icon  | `ReactNode`\|`false`\| `true`| false   | The icon of the alerts. It'd disappear if the value is false.                                        |
| indicator | `Indicator`              | —       | Indicator status of the avatar.                                                                      |
| css   | `CSS`                        | —       | The system prop that allows defining system overrides as well as additional CSS styles.              |

#### `<Avatar.Group/>` :

| Name     | Type                    | Default | Description                                                                             |
| :------- | :---------------------- | :------ | :-------------------------------------------------------------------------------------- |
| children | `ReactNode` \| `String` | —       | We pass our avatars as children props to Avatar.Group.                                  |
| size     | `xs`\|`sm`\|`md`\|`lg`  | —       | Sizes of the group.                                                                     |
| display  | `Number`                | —       | Display prop determines the number of avatar displayed.                                 |
| variant  | `contained`\|`outline`  | —       | Variant of the avatar on group.                                                         |
| css      | `CSS`                   | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

#### `<Avatar.Add/>` :

Extended all props of the Button

| Name  | Type                         | Default | Description                                                                                          |
| :---- | :--------------------------- | :------ | :--------------------------------------------------------------------------------------------------- |
| size  | `xs`\|`sm`\|`md`\|`lg`       | —       | Sizes of the button.                                                                                 |
| isDisabled | `boolean`               | —       | Whether the button is disabled                                                                       |
| css   | `CSS`                        | —       | The system prop that allows defining system overrides as well as additional CSS styles.              |

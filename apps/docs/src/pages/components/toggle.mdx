# Toggle

Provides the behavior and accessibility implementation for a toggle component. A toggle is similar to a checkbox, but represents on/off values as opposed to selection.

<Callout>
  Check out our [ladle](/ladle?story=toggle--variants) for more info on this
  component.
</Callout>
---

### Import:

```js
import {Toggle} from '@comfortdelgro/react-compass'
```

or

```js
import Toggle from '@comfortdelgro/react-compass/toggle'
```

---

### Basic Usage:

```js
<Toggle />
<Toggle isDisabled />
<Toggle isDisabled isSelected />
```

<section className={styles.avatarShowcaseContainer}>
  <Toggle />
  <Toggle isDisabled />
  <Toggle isDisabled isSelected />
</section>

### Custom styling:

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

<section className={styles.avatarShowcaseContainer}>
  <Toggle css={{'&:hover': {background: 'red'}}} />
</section>
```js
<Toggle css={{'&:hover': {background: 'red'}}} />
``` ---

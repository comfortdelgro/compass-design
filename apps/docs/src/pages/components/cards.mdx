# Cards

Cards provide a flexible and extensible content container with multiple variants and options

## Example

```tsx {22}
import {Card} from '@comfortdelgro/react-compass'

function BasicExample() {
  return (
    <Card size='sm'>
      <Card.Body>
        <Card.Title>Card title</Card.Title>
        <p>This is a card paragraph</p>
      </Card.Body>
    </Card>
  )
}
```

import {Card} from '@comfortdelgro/react-compass'
import ComponentDisplay from 'components/ComponentDisplay'

<ComponentDisplay>
  <Card size='sm'>
    <Card.Body>
      <Card.Title>Card title</Card.Title>
      <p style={{margin: 0}}>This is a card paragraph</p>
    </Card.Body>
  </Card>
</ComponentDisplay>

## Content

### Body

Use `<Card.Body>` to pad content inside a `<Card>`.

```tsx {22}
import {Card} from '@comfortdelgro/react-compass'

function BodyOnlyExample() {
  return (
    <Card size='sm'>
      <Card.Body>
        <p>This is a paragraph within a card body</p>
      </Card.Body>
    </Card>
  )
}
```

<ComponentDisplay>
  <Card size='sm'>
    <Card.Body>
      <p style={{margin: 0}}>This is a paragraph within a card body</p>
    </Card.Body>
  </Card>
</ComponentDisplay>

### Title

Use `<Card.Title>` to create a title inside a `<Card.Body>`.

```tsx {22}
import {Card} from '@comfortdelgro/react-compass'

function TitleExample() {
  return (
    <Card size='sm'>
      <Card.Body>
        <Card.Title>Card title</Card.Title>
        <p>This is a paragraph within a card body</p>
      </Card.Body>
    </Card>
  )
}
```

<ComponentDisplay>
  <Card size='sm'>
    <Card.Body>
      <Card.Title>Card title</Card.Title>
      <p style={{margin: 0}}>This is a paragraph within a card body</p>
    </Card.Body>
  </Card>
</ComponentDisplay>

You can also use a different heading tag inside `<Card.Title>`

```tsx {22}
import {Card} from '@comfortdelgro/react-compass'

function H1TitleExample() {
  return (
    <Card size='lg'>
      <Card.Body>
        <Card.Title>
          <h1>Card title as H1</h1>
        </Card.Title>
        <p>This is a paragraph within a card body</p>
      </Card.Body>
    </Card>
  )
}
```

<ComponentDisplay>
  <Card size='lg'>
    <Card.Body>
      <Card.Title>

      </Card.Title>
      <p style={{margin: 0}}>This is a paragraph within a card body</p>
    </Card.Body>
  </Card>
</ComponentDisplay>

### Action

Use `<Card.Action>` to create an area for action buttons in a card `<Card.Body>`.

```tsx {22}
import {Card} from '@comfortdelgro/react-compass'

function CardActionExample() {
  return (
    <Card size='lg'>
      <Card.Body>
        <Card.Title>Card title as H1</Card.Title>
        <p>This is a paragraph within a card body</p>
        <Card.Action>
          <Button variant='primary'>Button</Button>
          <Button variant='secondary'>Button</Button>
        </Card.Action>
      </Card.Body>
    </Card>
  )
}
```

```tsx {22}
    <Card size='lg'>
      <Card.Image src={imageSrc} />
      <Card.Body>
```

## Sizes

Card comes with different sizes: `sm`, `lg` and `full`.

```tsx {22}
import {Card} from '@comfortdelgro/react-compass'

function SizesExample() {
    const sizes = ['sm', 'lg', 'full']

    return (
       {sizes.map((size) => (
        <Card size={size}>
          <Card.Body>
            <Card.Title>
              Card with size: {size}
            </Card.Title>
            <p>This is a paragraph within a card body</p>
          </Card.Body>
        </Card>
       ))}
    );
}
```

<ComponentDisplay>
  {['sm', 'lg', 'full'].map((size) => (
    <Card key={size} css={{marginBottom: '24px'}} size={size}>
      <Card.Body>
        <Card.Title>Card with size</Card.Title>
        <p style={{margin: 0}}>This is a paragraph within a card body</p>
      </Card.Body>
    </Card>
  ))}
</ComponentDisplay>

You can fully customize the card size.

```tsx {22}
import {Card} from '@comfortdelgro/react-compass'

function CustomizedWidthExample() {
  const cardSize = '460px'

  return (
    <Card css={{width: cardSize}}>
      <Card.Body>
        <Card.Title>Card with customized size</Card.Title>
        <p>This is a paragraph within a card body</p>
      </Card.Body>
    </Card>
  )
}
```

<ComponentDisplay>
  <Card css={{width: '460px'}}>
    <Card.Body>
      <Card.Title> Card with customized size </Card.Title>
      <p style={{margin: 0}}>This is a paragraph within a card body</p>
    </Card.Body>
  </Card>
</ComponentDisplay>

## Disabled

Card can be disabled by passing `isDisabled` prop.

```tsx {22}
import {Card} from '@comfortdelgro/react-compass'
import Button from '@comfortdelgro/react-compass/button'

function DisabledCard() {
  const isDisabled = true
  return (
    <Card size='sm' isDisabled={isDisabled}>
      <Card.Body>
        <Card.Title>Disabled Card</Card.Title>
        <p>This is a paragraph within a card body</p>
        <Card.Action>
          <Button isDisabled={isDisabled} variant='primary'>
            Button
          </Button>
        </Card.Action>
      </Card.Body>
    </Card>
  )
}
```

import Button from '@comfortdelgro/react-compass/button'

<ComponentDisplay>
  <Card size='sm' isDisabled>
    <Card.Body>
      <Card.Title>Disabled Card</Card.Title>
      <p>This is a paragraph within a card body</p>
      <Card.Action>
        <Button isDisabled>Button</Button>
      </Card.Action>
    </Card.Body>
  </Card>
</ComponentDisplay>

## Clickable

Card can be clicked and focused by passing `isClickable` prop.

```tsx {22}
import {Card} from '@comfortdelgro/react-compass'

function DisabledCard() {
  return (
    <Card size='sm' isClickable onClick={() => alert('clicked')}>
      <Card.Body>
        <Card.Title>Clickable Card</Card.Title>
        <p>This is a paragraph within a card body</p>
      </Card.Body>
    </Card>
  )
}
```

<ComponentDisplay>
  <Card size='sm' isClickable onClick={() => alert('clicked')}>
    <Card.Body>
      <Card.Title>Clickable Card</Card.Title>
      <p>This is a paragraph within a card body</p>
      <Card.Action>
        <Button>Button</Button>
      </Card.Action>
    </Card.Body>
  </Card>
</ComponentDisplay>

## Shadowless

```tsx {22}
      <Card.Body>

}
```

## Props

### Card

| Name         | Type                   | Default | Description                                                                             |
| :----------- | :--------------------- | :------ | :-------------------------------------------------------------------------------------- |
| children     | `ReactNode`            | —       | The content to display in the Card.                                                     |
| size         | `sm` \| `lg` \| `full` | `full`  | The size of the Card                                                                    |
| isDisabled   | `boolean`              | false   | Whether the card is disabled.                                                           |
| isClickable  | `boolean`              | false   | Whether the card is clickable.                                                          |
| isShadowless | `boolean`              | false   | Whether the card is shadowless.                                                         |
| css          | `CSS`                  | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

### Card Body

| Name     | Type        | Default | Description                                                                             |
| :------- | :---------- | :------ | :-------------------------------------------------------------------------------------- |
| children | `ReactNode` | —       | The content to display in the Card Image.                                               |
| css      | `CSS`       | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

### Card Title

| Name     | Type        | Default | Description                                                                             |
| :------- | :---------- | :------ | :-------------------------------------------------------------------------------------- |
| children | `ReactNode` | —       | The content to display in the Card Image.                                               |
| css      | `CSS`       | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

### Card Action

| Name     | Type        | Default | Description                                                                             |
| :------- | :---------- | :------ | :-------------------------------------------------------------------------------------- |
| children | `ReactNode` | —       | The content to display in the Card Image.                                               |
| css      | `CSS`       | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

### Card Image

| Name   | Type     | Default | Description                                                                             |
| :----- | :------- | :------ | :-------------------------------------------------------------------------------------- |
| src \* | `string` | —       | The URL of the image.                                                                   |

| css    | `CSS`    | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

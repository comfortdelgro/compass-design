### Default

<Toast color='neutral' />

```js
<Button
    css={{width: '8rem', color: 'black'}}
    onPress={() => setIsOpen(true)}
>
    Open toast
</Button>
    <Toast
        isOpen={isOpen}
        handleClose={() => setIsOpen(false)}
    >
        <Toast.Icon>
          <Icon icon={faImage} />
        </Toast.Icon>
        <Toast.Title>My Title</Toast.Title>
        <Toast.Label>a few minutes ago</Toast.Label>
        <Toast.CloseIcon>
          <Icon icon={faXmark} />
        </Toast.CloseIcon>
        <Toast.Message>Lorem ipsum dolor, sit amet consectetur.</Toast.Message>
        <Toast.Actions>
          <Button css={{color: 'black'}}>Take action</Button>
          <Button css={{color: 'black'}}>Take action</Button>
        </Toast.Actions>
    </Toast>
```

## Props

| Name        | Type                                                   | Default   | Description                                                                             |
| :---------- | :----------------------------------------------------- | :-------- | :-------------------------------------------------------------------------------------- |
| color       | `informative` \| `neutral` \| `negative` \| `positive` | 'neutral' | Variants                                                                                |
| handleClose | `Function`                                             | -         | Callback to close toast.                                                                |
| isOpen      | `boolean`                                              | false     | The toast is visible when true.                                                         |
| autoClose   | `false` \| `number`                                    | false     | Number of miliseconds to auto close.                                                    |
| css         | `CSS`                                                  | â€”         | The system prop that allows defining system overrides as well as additional CSS styles. |

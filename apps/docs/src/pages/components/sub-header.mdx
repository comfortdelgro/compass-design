# SubHeader

The Sub header is then used to follow up on the top of subsequent child pages

---

### Import

```js
import {SubHeader} from '@comfortdelgro/react-compass'
```

or

```js
import SubHeader from '@comfortdelgro/react-compass/page-header'
```

---

### Basic SubHeader

<section className={styles.componentShowcaseContainer}>
  <SubHeaderComponent />
</section>

```js
<SubHeader>
  <SubHeader.Header>
    <SubHeader.Title>Title</SubHeader.Title>
  </SubHeader.Header>
  <SubHeader.Description>Description.</SubHeader.Description>
</SubHeader>
```

# Popover

When you want to show a content on above and anchor with an element when user click on it, just use this component follow this guidelines.

This component can be used for some cases like notification list, user menu.

Actually, you can put whatever you want into the popover content, which is the children of Popover. Default display of popover is inline-block to keep it wraps around the anchor element.

---

### Props:

| Name      | Type                                                                             | Default        | Description                                        |
| :-------- | :------------------------------------------------------------------------------- | :------------- | :------------------------------------------------- |
| anchor    | `React.ReactNode`                                                                | `-`            | Element that when you click on to show the Popover |
| isOpen    | `boolean`                                                                        | `-`            | Let Popover content open                           |
| direction | `bottom-right` `bottom-left` `bottom-center` `top-right` `top-left` `top-center` | `bottom-right` | Direction of Popover content                       |

---

### Methods:

| Name    | Type         | Default                                       | Description |
| :------ | :----------- | :-------------------------------------------- | :---------- |
| onClose | `() => void` | Event that calls when use has clicked outside |

---

### Import:

```js
import Popover, {PopoverDirection} from '@comfortdelgro/react-compass/popover'
```

# Link

The Link component allows you to easily customize anchor elements with your theme colors and typography styles.

---

### Import:

```js
import {Link} from '@comfortdelgro/react-compass'
```

or

```js
import Link from '@comfortdelgro/react-compass/link'
```

---

### Basic Link:

<section className={styles.avatarShowcaseContainer}>
  <Link href='/#page'>Go to another page</Link>
  <Link href='https://google.com' external>
    Go to Google
  </Link>
</section>

```js
<Link href='/#page'>Go to another page</Link>
<Link href='https://google.com' external>
  Go to Google
</Link>
```

---

### Custom styling:

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

<section className={styles.avatarShowcaseContainer}>
  <Link
    href='https://google.com'
    external
    css={{textDecoration: 'line-through', '&:hover': {color: 'grey'}}}
  >
    Go to Google
  </Link>
</section>

```js
<Link
  href='https://google.com'
  external
  css={{textDecoration: 'line-through', '&:hover': {color: 'grey'}}}
>
  Go to Google
</Link>
```

# Transitions

The Transitions component provides you various transition effects.

<Callout>
  You can see how this component is implemented on
  [ladle](https://comfortdelgro.github.io/compass-design/ladle/?story=transitions--variants)
</Callout>
---

### Import:

```js
import {Box} from '@comfortdelgro/react-compass'
```

or

```js
import Box from '@comfortdelgro/react-compass/box'
```

---

### The 'as' prop:

The 'as' prop refers to the HTML element that the Box component should render as. By default, the Box component will render as a div element, but if you pass a different value to the as prop, it will render as that element instead.

<section className={styles.bannerShowcaseContainer}>
<div>
  <Box
    css={{
      backgroundColor: 'grey',
      width: '8rem',
      textAlign: 'center',
      height: '2rem',
      textAlign: 'center',
      paddingTop: '0.3rem',
    }}
  >
    <p>My 'div' box</p>
  </Box>
</div>
<div>
    <Box as='h2' color='red'>
        <p>My 'h2' box</p>
    </Box>

</div>
</section>

```js
<Box
  css={{
    backgroundColor: 'grey',
    width: '8rem',
    textAlign: 'center',
    height: '2rem',
    textAlign: 'center',
    paddingTop: '0.3rem',
  }}
>
  <p>My 'div' box</p>
</Box>

<Box as='h2' color='red'>
    <p>My 'h2' box</p>
</Box>
```

---

### System props you can pass the Box component:

| Name         | Type   | Default | Description |
| ------------ | ------ | ------- | ----------- |
| border       | string |         |             |
| borderBottom | string |         |             |
| borderColor  | string |         |             |
| borderLeft   | string |         |             |
| borderRadius | string |         |             |
| borderRight  | string |         |             |
| borderTop    | string |         |             |
| boxShadow    | string |         |             |
| displayPrint | string |         |             |
| displayRaw   | string |         |             |

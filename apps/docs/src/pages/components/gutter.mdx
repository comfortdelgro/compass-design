# Gutter Component

This gutter control will align with parent element, it will update directly to parent element size.
To implement it, just place it into the element that you want to resize.

<Callout>
  Check out our [ladle](/ladle?story=gutter--default) for more info on this
  component.
</Callout>

---

### Import:

```js
import {Grid, GutterContent, GutterResizer} from '@comfortdelgro/react-compass'
```

or

```js
import Gutter, {
  GutterContent,
  GutterResizer,
} from '@comfortdelgro/react-compass/gutter'
```

### Props

#### Gutter

| Name         | Type               | Default | Description                                                  |
| :----------- | :----------------- | :------ | :----------------------------------------------------------- |
| children     | `React.ReactNode`  | —       |                                                              |
| css          | `CSS`              | —       |                                                              |
| width        | `CSS['width']`     | —       | initial width of gutter                                      |
| minWidth     | `CSS['min-width']` | —       | the minimum width when resizing                              |
| maxWidth     | `CSS['max-width']` | —       | the maximum width when resizing                              |
| hasBothSides | `boolean`          | `false` | a prop that helps us detect if there are two sides of gutter |

#### Gutter Resizer

| Name               | Type                             | Default | Description                                     |
| :----------------- | :------------------------------- | :------ | :---------------------------------------------- |
| css                | `CSS`                            | —       |                                                 |
| showCollapseButton | `boolean`                        | `false` | show collapse button                            |
| isCollapsed        | `boolean`                        | —       | controlled state                                |
| minCollapseWidth   | `CSS['min-width']`               | —       | the minimum width when trigger collapse button  |
| maxCollapseWidth   | `CSS['max-width']`               | —       | the maximum width when trigger collapse button  |
| onCollapseChange   | `(isCollapsed: boolean) => void` | —       | a callback that fire when collapse state change |

#### Gutter Content

| Name     | Type              | Default | Description |
| :------- | :---------------- | :------ | :---------- |
| children | `React.ReactNode` | —       |             |
| css      | `CSS`             | —       |             |

# Carousel

A group of slider components. The core component is ContentSlider with common feature is switching slides. It allows you to pass a list of children elements, to keep it flexible.
Children should be `CarouselSlide` or `CarouselImageSlide`. In case you want to customize it, please use `slider-side` class name for children and `active` class name for slide that activated.

For next version of carousel, we have CarouselV2, this component extends and reuse ContentSlider component. To use it, you just need to simply pass the array of data into it.

The design for mobile mode is too different with desktop mode, we can not handle it by using the same template, so now we have CarouselMobile component. Otherwise, ContentSlider will have basic responsive layout itself.

---

### Components:

| Name               | Type        | Default | Description                                                                      |
| :----------------- | :---------- | :------ | :------------------------------------------------------------------------------- |
| ContentSlider      | `Component` |         | Core slider that can be reused for multiple purpose of sliding things            |
| CarouselV2         | `Component` |         | Next version of current carousel that extends from ContentSlider                 |
| CarouselMobile     | `Component` |         | Sepecial extended version of ContentSlider to support different layout on mobile |
| CarouselSlide      | `Component` |         | A component that contains slide content                                          |
| CarouselImageSlide | `Component` |         | Slide component use for a slider that accept only images                         |

### Types:

| Name                | Type                  | Default | Description                                                                                  |
| :------------------ | :-------------------- | :------ | :------------------------------------------------------------------------------------------- |
| ContentSliderItem   | `ContentSliderItem`   |         | Promotion slider item data                                                                   |
| SocicalIcon         | `SocicalIcon`         |         | Item contains social icon data: `icon` and `url`                                             |
| NinePartAlignment   | `NinePartAlignment`   |         | Alignment that apply for a single slide item, remember to put it into ContentSliderItem data |
| ContentSliderButton | `ContentSliderButton` |         | Button that use in sliders definition: `type`, `label`, `callback`                           |
| ButtonType          | `ButtonType`          |         | `primary` `secondary`                                                                        |

---

### Props:

| Name                 | Type                    | Default | Description                                                                              |
| :------------------- | :---------------------- | :------ | :--------------------------------------------------------------------------------------- |
| autoSwitch           | `boolean`               | `true`  | To let the slider auto switch in every 300ms                                             |
| useNavigation        | `boolean`               | `true`  | To show next and previous buttons                                                        |
| useDotIndicator      | `boolean`               | `true`  | To show dot indicators                                                                   |
| navigationButtonType | `icon` `text`           | `icon`  | To use arrow icon or next & prev text                                                    |
| data                 | `ContentSliderItem[]`   | `-`     | Required parameters                                                                      |
| socials              | `SocicalIcon[]`         | `[]`    | List of social icons definition                                                          |
| buttons              | `ContentSliderButton[]` | `[]`    | Buttons that displays under the slider, so currently it only works with `CarouselMobile` |

---

### Methods:

| Name          | Type                      | Default  | Description                              |
| :------------ | :------------------------ | :------- | :--------------------------------------- |
| onSwitchSlide | `(index: number) => void` | `number` | Event that calls when slide has switched |

### Image only slider:

<section className={styles.contentSliderDemo}>
  <ImageOnlyDemo />
</section>

```js
const [activeIndex, setActiveIndex] = useState(0)

const imageUrls = [
  'https://mdbcdn.b-cdn.net/img/Photos/Slides/img%20(15).webp',
  'https://mdbcdn.b-cdn.net/img/Photos/Slides/img%20(22).webp',
  'https://mdbcdn.b-cdn.net/img/Photos/Slides/img%20(23).webp',
]

const handleSwitchSlide = (index: number) => {
  setActiveIndex(index)
}

return (
  <ContentSlider onSwitchSlide={handleSwitchSlide}>
    {imageUrls.map((imageUrl: string, index: number) => (
      <CarouselImageSlide
        key={index}
        active={activeIndex === index}
        imageUrl={imageUrl}
      />
    ))}
  </ContentSlider>
)
```

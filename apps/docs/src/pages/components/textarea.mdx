# Text areas

Progress indicators commonly known as spinners, express an unspecified wait time or display the length of a process.

---

### Import:

```js
import {Textarea} from '@comfortdelgro/react-compass'
```

or

```js
import Textarea from '@comfortdelgro/react-compass/textarea'
```

We recommend the second method to import components for the benefits of [tree shaking](https://developer.mozilla.org/en-US/docs/Glossary/Tree_shaking).

### Custom styling:

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

<section className={styles.avatarShowcaseContainer}>
  <Textarea
    label='Text Label'
    placeholder='Enter your message'
    wordCount
    maxLength={50}
    errored
    css={{height: '10em', width: '50em'}}
  />
</section>

```js
<Textarea
  label='Text Label'
  placeholder='Enter your message'

  maxLength={50}

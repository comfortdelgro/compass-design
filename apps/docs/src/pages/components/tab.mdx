# Tab

Tab can do quick switch between your content.

---

### Import:

```js
import {Tab, TabProps} from '@comfortdelgro/react-compass'
```

or

```js
import Tab, {TabProps} from '@comfortdelgro/react-compass/tab'
```

We recommend the second method to import components for the benefits of [tree shaking](https://developer.mozilla.org/en-US/docs/Glossary/Tree_shaking).

### Custom styling:

You can style React Compass's components with the css prop.
With css prop, you can basically do anything that css can do.
```js
<Tab css={{opacity: '0.8', '&:hover': {opacity: '0.5', cursor: 'pointer'}}}>
	<Tab.List>
		<Tab.Item value={1} icon="top">Tab1</Tab.Item>
		<Tab.Item value={2} icon="left">Tab2</Tab.Item>
		<Tab.Item value={3} icon="right">Tab3</Tab.Item>
		<Tab.Item key='4' value={4} icon="none" isDisabled>Tab3</Tab.Item>
	</Tab.List>
	<Tab.Panel value={1}>Tab1 content</Tab.Panel>
	<Tab.Panel value={2}>Tab2 content</Tab.Panel>
	<Tab.Panel value={3}>Tab3 content</Tab.Panel>
</Tab>
```

### Props:

#### `<Tab/>` :

| Name               | Type                   | Default     | Description                                                                                              |
| :----------------- | :--------------------- | :---------- | :------------------------------------------------------------------------------------------------------- |
| children*          | `ReactNode`            | —           | The contents of the collection.                                                                          |
| variant            | `rounded`,`simple`     | `rounded`   | Style of the tab                                                                                         |
| orientation        | `horizontal`,`vertical`| `horizontal`| Direction of the tab |                                                                                   |
| isDisabled         | `boolean`              | `false`     | Whether the Tabs are disabled. Shows that a selection exists, but is not available in that circumstance. |
| disabledKeys       | `Key[]`                | —           | The item keys that are disabled. These items cannot be selected, focused, or otherwise interacted with.  |
| defaultSelectedKey | `Key`                  | —           | The currently selected key in the collection (uncontrolled).                                             |
| selectedKey        | `Key`                  | —           | The currently selected key in the collection (controlled).                                               |
| onSelectionChange  | `(value) => void`      | —           | Handler that is called when the selection changes.                                                       |
| className          | `String`               | —           | Extra class name for tab.                                                                                |
| css                | `CSS`                  | —           | The system prop that allows defining system overrides as well as additional CSS styles.                  |

#### `<Tab.List/>` :

| Name       | Type                    | Default | Description                                                                             |
| :--------- | :---------------------- | :------ | :-------------------------------------------------------------------------------------- |
| children*  | `ReactNode` \| `String` | —       | We pass our avatars as children props to Tab.Item.                                      |
| className  | `String`                | —       | Extra class name for tab                                                                |
| css        | `CSS`                   | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

#### `<Tab.Item/>` :

| Name        | Type                               | Default | Description                                                                             |
| :---------- | :--------------------------------- | :------ | :-------------------------------------------------------------------------------------- |
| icon        | `none` \| `left`\| `right`\| `top` | `none`  | Display icon in tab content                                                             |
| isDisabled  | `Boolean`                          | `false` | Status of the Item                                                                      |
| value*      | `Key`                              | —       | Value of the Item                                                                       |
| key         | `Key`                              | —       | Key of the Item                                                                         |
| children    | `ReactNode` \| `String`            | —       | Content of the tab Item                                                                 |
| className   | `String`                           | -       | Extra class name for tab                                                                |
| css         | `CSS`                              | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

#### `<Tab.Panel/>` :

| Name        | Type                    | Default | Description                                                                             |
| :---------- | :---------------------- | :------ | :-------------------------------------------------------------------------------------- |
| key         | `Key`                   | —       | Key of the display content
| value*      | `Key`                   | -       | Value of the display content
| children*   | `ReactNode` \| `String` | —       | Content will be display with matched value with Tab.Item                                                                      
| className   | `String`                | —       | Extra class name for tab                  
| css         | `CSS`                   | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

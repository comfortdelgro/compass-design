# Sidenav

The Sidenav displays horizontally information and navigation relating to the current screen.

---

### Import

```js
import {Sidenav} from '@comfortdelgro/react-compass'
```

or

```js
import Sidenav from '@comfortdelgro/react-compass/Sidenav'
```

---

### Expand

You can expand the Sidenav without hovering

<section className={styles.componentShowcaseContainer}>
  <Sidenav css={{height: '800px'}} expand>
    <Sidenav.Item isActive>
      <Icon icon={faHelicopter}></Icon>
      <span>Menu</span>
    </Sidenav.Item>
  </Sidenav>
</section>

```js {1}
<Sidenav css={{height: '800px'}} expand>
  <Sidenav.Item isActive>

  </Sidenav.Item>
</Sidenav>
```

### Sidenav Item

Sidenav Item accepts two children. If Sidenav is not in expand mode, only the first child will be rendered

<section className={styles.componentShowcaseContainer}>
  <Sidenav css={{height: '800px'}}>
    <Sidenav.Item isActive>
      <Icon icon={faHelicopter}></Icon>
      <span>Menu</span>
    </Sidenav.Item>
  </Sidenav>
</section>

```js {2-5}
<Sidenav css={{height: '800px'}}>
  <Sidenav.Item isActive>
    <Icon icon={faHelicopter}></Icon>
    <span>Menu</span>
  </Sidenav.Item>
</Sidenav>
```

### Props

#### Sidenav

| Name     | Type              | Default | Description                                                     |
| :------- | :---------------- | :------ | :-------------------------------------------------------------- |
| children | `React.ReactNode` | —       |                                                                 |
| expand   | `boolean`         | false   | Whether the Sidenav is expanded by default or expanded by hover |
| css      | `CSS`             | —       |                                                                 |

#### Sidenav Item

| Name       | Type              | Default | Description                                    |
| :--------- | :---------------- | :------ | :--------------------------------------------- |
| children\* | `React.ReactNode` | —       | Accept only two children                       |
| isActive   | `boolean`         | false   | Whether the item is currently activated or not |
| css        | `CSS`             | —       |                                                |

#### Sidenav Divider

| Name | Type  | Default | Description |
| :--- | :---- | :------ | :---------- |
| css  | `CSS` | —       |             |

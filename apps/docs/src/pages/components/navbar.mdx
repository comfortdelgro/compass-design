# Navbar

The Navbar displays horizontally information and navigation relating to the current screen.

<Callout>
  Check out our [ladle](/ladle?story=navbar--variants) for more info on this
  component.
</Callout>

---

### Import

```js
import {Navbar} from '@comfortdelgro/react-compass'
```

or

```js
import Navbar from '@comfortdelgro/react-compass/navbar'
```

---

### Brand

Navbar Brand renders heading text that adapt to the current [color](#colors) of the Navbar

<section className={styles.componentShowcaseContainer}>
  <Navbar>
    <Navbar.Brand>Workbench Title</Navbar.Brand>
  </Navbar>
</section>

```js
<Navbar>
  <Navbar.Brand>Workbench Title</Navbar.Brand>
</Navbar>
```

      </a>

      </a>

  </Navbar>
</section>

### Actions

Navbar Actions helps organize action buttons. The prop `alternativeElement` provides the alternate action button when the screen size is small

<section className={styles.componentShowcaseContainer}>
  <Navbar>
    <Navbar.Brand>Workbench Title</Navbar.Brand>
    <Navbar.Links>
      <a href='#'>
        <Button>Home</Button>
      </a>
      <a href='#'>
        <Button>Services</Button>
      </a>
      <a href='#'>
        <Button>Products</Button>
      </a>
    </Navbar.Links>
    <Navbar.Separator></Navbar.Separator>
    <Navbar.Actions alternativeElement={<DocumentIcon color='white' />}>
      <QuestionIcon color='white'></QuestionIcon>
      <NotificationIcon color='white'></NotificationIcon>
      <OrgChatIcon color='white'></OrgChatIcon>
      <Avatar label='Avatar' size='xs' />
    </Navbar.Actions>
  </Navbar>
</section>

```js
<Navbar>
  <Navbar.Brand>Workbench Title</Navbar.Brand>
  <Navbar.Links>
    <a href='#'>
      <Button>Home</Button>
    </a>
    <a href='#'>
      <Button>Services</Button>
    </a>
    <a href='#'>
      <Button>Products</Button>
    </a>
  </Navbar.Links>
  <Navbar.Separator></Navbar.Separator>
  <Navbar.Actions alternativeElement={<DocumentIcon color='white' />}>
```

### Props

#### Navbar

| Name     | Type                  | Default | Description |
| :------- | :-------------------- | :------ | :---------- |
| children | `React.ReactNode`     | —       |             |
| color    | `white` \| `blue`     | —       |             |
| variant  | `portal` \| `website` | —       |             |
| css      | `CSS`                 | —       |             |

#### Navbar.Brand

| Name           | Type              | Default | Description                                                   |
| :------------- | :---------------- | :------ | :------------------------------------------------------------ |
| children       | `React.ReactNode` | —       |                                                               |
| hiddenOnMobile | `boolean`         | —       | Brand name will be hidden when screen is resize to small size |
| css            | `CSS`             | —       |                                                               |

#### Navbar.Links

| Name     | Type              | Default | Description |
| :------- | :---------------- | :------ | :---------- |
| children | `React.ReactNode` | —       |             |
| css      | `CSS`             | —       |             |

#### Navbar.Separator

| Name     | Type              | Default | Description |
| :------- | :---------------- | :------ | :---------- |
| children | `React.ReactNode` | —       |             |
| css      | `CSS`             | —       |             |

#### Navbar.Actions

| Name               | Type              | Default | Description                                                         |
| :----------------- | :---------------- | :------ | :------------------------------------------------------------------ |
| children           | `React.ReactNode` | —       |                                                                     |
| alternativeElement | `React.ReactNode` | —       | Alternative element to show when the screen is resize to small size |
| css                | `CSS`             | —       |                                                                     |

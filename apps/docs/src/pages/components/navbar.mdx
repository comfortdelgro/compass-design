# Navbar

The Navbar displays horizontally information and navigation relating to the current screen.

---

### Import

```js
import {Navbar} from '@comfortdelgro/react-compass'
```

or

```js
import Navbar from '@comfortdelgro/react-compass/navbar'
```

---

### Basic Navbar

<section className={styles.componentShowcaseContainer}>
  <Navbar>
    <Avatar size='sm' label='asd' />
    <Navbar.Brand>Workbench Title</Navbar.Brand>
    <Button>Home</Button>
    <Button>Services</Button>
    <Button>Products</Button>
  </Navbar>
</section>


### Brand

Navbar Brand renders heading text that adapt to the current [color](#colors) of the Navbar

<section className={styles.componentShowcaseContainer}>
  <Navbar>
    <Navbar.Brand>Workbench Title</Navbar.Brand>
  </Navbar>
</section>

```js
<Navbar>
  <Navbar.Brand>Workbench Title</Navbar.Brand>
</Navbar>
```

### Icon

Navbar Icon renders Icon that adapt to the current [color](#colors) of the Navbar

<section className={styles.componentShowcaseContainer}>
  <Navbar>
    <Navbar.Icon icon={faSearch} />
  </Navbar>
</section>

```js
<Navbar>
  <Navbar.Icon icon={faSearch} />
</Navbar>
```

# Sidebar

The Sidebar displays horizontally information and navigation relating to the current screen.

---

### Import

```js
import {Sidebar} from '@comfortdelgro/react-compass'
```

or

```js
import Sidebar from '@comfortdelgro/react-compass/sidebar'
```

---

### Expand

You can expand the sidebar without hovering

<section className={styles.componentShowcaseContainer}>
  <Sidebar css={{height: '800px'}} expand>
    <Sidebar.Item isActive>
      <Icon icon={faHelicopter}></Icon>
      <span>Menu</span>
    </Sidebar.Item>
  </Sidebar>
</section>

```js {1}
<Sidebar css={{height: '800px'}} expand>
  <Sidebar.Item isActive>
    <Icon icon={faHelicopter}></Icon>
    <span>Menu</span>
  </Sidebar.Item>
</Sidebar>
```

### Sidebar Item

Sidebar Item accepts two children. If sidebar is not in expand mode, only the first child will be rendered

<section className={styles.componentShowcaseContainer}>
  <Sidebar css={{height: '800px'}}>
    <Sidebar.Item isActive>
      <Icon icon={faHelicopter}></Icon>
      <span>Menu</span>
    </Sidebar.Item>
  </Sidebar>
</section>

```js {2-5}
<Sidebar css={{height: '800px'}}>
  <Sidebar.Item isActive>
    <Icon icon={faHelicopter}></Icon>
    <span>Menu</span>
  </Sidebar.Item>
</Sidebar>
```

### Props

#### Sidebar

| Name     | Type              | Default | Description                                                     |
| :------- | :---------------- | :------ | :-------------------------------------------------------------- |
| children | `React.ReactNode` | —       |                                                                 |
| expand   | `boolean`         | false   | Whether the sidebar is expanded by default or expanded by hover |
| css      | `CSS`             | —       |                                                                 |

#### Sidebar Item

| Name       | Type              | Default | Description                                    |
| :--------- | :---------------- | :------ | :--------------------------------------------- |
| children\* | `React.ReactNode` | —       | Accept only two children                       |
| isActive   | `boolean`         | false   | Whether the item is currently activated or not |
| css        | `CSS`             | —       |                                                |

#### Sidebar Divider

| Name | Type  | Default | Description |
| :--- | :---- | :------ | :---------- |
| css  | `CSS` | —       |             |

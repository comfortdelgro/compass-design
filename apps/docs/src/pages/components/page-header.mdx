# PageHeader, SubHeader

The Page header is used to begin pages and is a format guideline to how the top of a page should be organised

---

### Import

```js
import {PageHeader} from '@comfortdelgro/react-compass'
import {PageHeader} from '@comfortdelgro/react-compass'
```

or

```js
import PageHeader from '@comfortdelgro/react-compass/page-header'
import PageHeader from '@comfortdelgro/react-compass/page-header'
```

---

### PageHeader Colors

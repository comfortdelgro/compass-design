# Divider

The Divider component allows you to easily customize anchor elements with your theme colors and typography styles.

---

### Import:

```js
import {Divider} from '@comfortdelgro/react-compass'
```

or

```js
import Divider from '@comfortdelgro/react-compass/divider'
```

---

### Basic Divider:

<section className={styles.avatarShowcaseContainer} style={{flexDirection: 'column'}}>
  <Divider />
  <div style={{height: 200}}>
    <Divider orientation='vertical' />
  </div>
</section>

```js
<Divider />
<Divider orientation='vertical' />
```

---

### Align Divider:

<section className={styles.avatarShowcaseContainer} style={{flexDirection: 'column'}}>
  <Divider align='left'>
    <span>Left</span>
  </Divider>
  <Divider align='center'>
    <span>Center</span>
  </Divider>
  <Divider align='right'>
    <span>Right</span>
  </Divider>
</section>

```js
<Divider align='left'>
  <span>Left</span>
</Divider>
<Divider align='center'>

</Divider>
```

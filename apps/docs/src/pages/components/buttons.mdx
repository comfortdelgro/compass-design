# Button

Buttons allow users to take actions, and make choices, with a single tap.

---

### Import:

```js
import {Button} from '@comfortdelgro/react-compass'
```

or

```js
import Button from '@comfortdelgro/react-compass/button'
```

### Custom styling:

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

<section className={styles.avatarShowcaseContainer}>
  <Button
    variant='danger'
    rightIcon={<Icon icon={faThumbsUp} />}
    css={{'&:hover': {color: 'grey'}}}
  >
    Danger
  </Button>
</section>

```js
<Button
  variant='danger'
  rightIcon={<Icon icon={faThumbsUp} />}
  css={{'&:hover': {color: 'grey'}}}
>
  Danger
</Button>
```

---

### Props:

| Name       | Type                                            | Default | Description |
| :--------- | :---------------------------------------------- | :------ | :---------- |
| color      | `info` \| `success` \| `warning` \| `danger`    |         |             |
| rightIcon  | `IconProp`\|`false`\| `true`                    | —       |             |
| leftIcon   | `IconProp`\|`false`\| `true`                    |         |             |
| variant    | `primary` \| `secondary` \| `danger` \| `ghost` |         |             |
| css        | `CSS`                                           | —       |             |
| isDisabled |                                                 |         |             |
| loading    | `false`\| `true`                                |         |             |

# Tooltips

Display container for Tooltip content. Has a directional arrow dependent on its placement.

---

### Import

```js
import {Tooltip, TooltipTrigger} from '@comfortdelgro/react-compass'
```

or

```js
import {Tooltip, TooltipTrigger} from '@comfortdelgro/react-compass/tooltip'
```

We recommend the second method to import components for the benefits of [tree shaking](https://developer.mozilla.org/en-US/docs/Glossary/Tree_shaking).

---

### Content

The TooltipTrigger accepts exactly two children: the element which triggers the display of the Tooltip and the Tooltip itself. The trigger element must be the first child passed into the TooltipTrigger. All content should be internationalized.

### Basic tooltip

```js
<TooltipTrigger>
  <a href="#">Hover me</a>
  <Tooltip>
    Tooltip
  </Tooltip>
</TooltipTrigger>
```
<ComponentDisplay>
  <TooltipTrigger>
    <a href='#'>Hover me</a>
    <Tooltip>
      Tooltip
    </Tooltip>
  </TooltipTrigger>
</ComponentDisplay>

### Title

Render title of the tooltip

```js
<TooltipTrigger>
  <a href='#'>Hover me</a>
  <Tooltip title="Title" dismissible>
    Tooltip
  </Tooltip>
</TooltipTrigger>
```

<ComponentDisplay>
  <TooltipTrigger>
    <a href='#'>Hover me</a>
    <Tooltip title={<p style={{fontSize: '20px', margin: 0}}>Title</p>} dismissible>
      <p style={{marginTop: 0}}>Tooltip</p>
    </Tooltip>
  </TooltipTrigger>
</ComponentDisplay>

### Tooltip Delay

Adjust delay of tooltip after hovering the trigger

```js
<TooltipTrigger delay={1000}>
  <a href='#'>Hover me</a>
  <Tooltip>
    Tooltip
  </Tooltip>
</TooltipTrigger>
```
<ComponentDisplay>
  <TooltipTrigger delay={1000}>
    <a href='#'>Hover me</a>
    <Tooltip>
      Tooltip
    </Tooltip>
  </TooltipTrigger>
</ComponentDisplay>

### Custom styling

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

```js
<TooltipTrigger>
  <a href='#'>Hover me</a>
  <Tooltip
    title={<p style={{fontSize: '20px', margin: 0}}>Title</p>}
    dismissible 
    css={{backgroundColor: 'black'}}
  >
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Recusandae
    blanditiis laborum commodi mollitia officiis minus ipsum totam fuga
    odit, dicta magnam numquam reprehenderit et sequi dignissimos iusto
    deleniti. Quia, modi
  </Tooltip>
</TooltipTrigger>
```

<ComponentDisplay>
  <TooltipTrigger>
    <a href='#'>Hover me</a>
    <Tooltip
      title={<p style={{fontSize: '20px', margin: 0}}>Title</p>}
      dismissible 
      css={{backgroundColor: 'black'}}
    >
     <p style={{margin: '16px 0'}}>Lorem ipsum dolor sit amet consectetur adipisicing elit. Recusandae
      blanditiis laborum commodi mollitia officiis minus ipsum totam fuga
      odit, dicta magnam numquam reprehenderit et sequi dignissimos iusto
      deleniti. Quia, modi</p>
    </Tooltip>
  </TooltipTrigger>
</ComponentDisplay>

### Placement

The Tooltip's placement with respect to its trigger element can be adjusted using the placement prop. See the props table below for a full list of available placement combinations

```js
<TooltipTrigger placement="right">
  <a href='#'>Hover me</a>
  <Tooltip>
    Open on the right
  </Tooltip>
</TooltipTrigger>
```

<ComponentDisplay>
  <TooltipTrigger placement="right">
    <a href='#'>Hover me</a>
    <Tooltip>
      <p style={{marginTop: 0}}>Open on the right</p>
    </Tooltip>
  </TooltipTrigger>
</ComponentDisplay>

### Offset

```js
<TooltipTrigger>
  <a href='#'>Hover me</a>
  <Tooltip>
    This will shift down.
  </Tooltip>
</TooltipTrigger>
```

<ComponentDisplay>
  <TooltipTrigger offset={50}>
    <a href='#'>Hover me</a>
    <Tooltip>
      <p style={{marginTop: 0}}> This will shift down.</p>
    </Tooltip>
  </TooltipTrigger>
</ComponentDisplay>

```js
<TooltipTrigger>
  <a href='#'>Hover me</a>
  <Tooltip>
    This will shift over to the right.
  </Tooltip>
```

    <Tooltip>
      <p style={{marginTop: 0}}>This will shift over to the right.</p>

  </TooltipTrigger>

## Props

### TooltipTrigger Props:

| Name         | Type                             | Default  | Description                                                                                                                                        |
| :----------- | :------------------------------- | :------- | :------------------------------------------------------------------------------------------------------------------------------------------------- |
| children\*   | [`ReactElement`, `ReactElement`] | —        | The first child must be a trigger, the second child must be Tooltip component                                                                      |
| offset       | `number`                         | 16       | The additional offset applied along the main axis between the element and its anchor element.                                                      |
| crossOffset  | `number`                         | 0        | The additional offset applied along the cross axis between the element and its anchor element.                                                     |
| delay        | `number`                         | 0        | The delay time for the tooltip to show up                                                                                                          |
| trigger      | `focus`                          | —        | By default, opens for both focus and hover. Can be made to open only for focus.                                                                    |
| open         | `boolean`                        | —        | Whether the overlay is open by default (controlled)                                                                                                |
| defaultOpen  | `boolean`                        | —        | Whether the overlay is open by default (uncontrolled).                                                                                             |
| placement    | [`Placement`](#placement-prop)   | `bottom` | The placement of the element with respect to its anchor element.                                                                                   |
| shouldFlip   | `boolean`                        | `true`   | Whether the element should flip its orientation (e.g. top to bottom or left to right) when there is insufficient room for it to render completely. |
| onOpenChange | `(isOpen: boolean) => void`      | —        | Handler that is called when the overlay's open state changes.                                                                                      |
| css          | `CSS`                            | —        | The system prop that allows defining system overrides as well as additional CSS styles.                                                            |

#### Placement Prop

| Name      | Type                                                                                                                                                                                                                                                                                                                   | Default | Description                                                      |
| :-------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------ | :--------------------------------------------------------------- |
| placement | `bottom` \| `bottom left` \| `bottom right` \| `bottom start` \| `bottom end` \| `top` \| `top left` \| `top right` \| `top start` \| `top end` \| `left` \| `left top` \| `left bottom` \| `start` \| `start top` \| `start bottom` \| `right` \| `right top` \| `right bottom` \| `end` \| `end top` \| `end bottom` | —       | The placement of the element with respect to its anchor element. |

### Tooltip Props

| Name        | Type                    | Default | Description                                                                             |
| :---------- | :---------------------- | :------ | :-------------------------------------------------------------------------------------- |
| children\*  | `ReactNode`             | —       |                                                                                         |
| title       | `ReactNode` \| `string` | —       | Render tooltip title                                                                    |
| dismissible | `boolean`               | `false` | If true, show the close icon.                                                           |
| css         | `CSS`                   | —       | The system prop that allows defining system overrides as well as additional CSS styles. |

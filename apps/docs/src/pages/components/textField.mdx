# Text Field

Text Fields let users enter and edit text.

---

### Import:

```js
import {Textfield} from '@comfortdelgro/react-compass'
```

or

```js
import Textfield from '@comfortdelgro/react-compass/textfield'
```

---

<section className={styles.avatarShowcaseContainer}>
  <Textfield placeholder='Enter your name' />
</section>


### Custom styling:

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

<section className={styles.avatarShowcaseContainer}>
  <Textfield
    value='Wrong Value'
    label='Name'
    placeholder='Enter your name'
    helperText='Use Helper Text for Errors'
    isDisabled
    css={{width: '20rem'}}
  />
</section>

```js
<Textfield
  value='Wrong Value'
  label='Name'
  placeholder='Enter your name'
  helperText='Use Helper Text for Errors'
  isDisabled
  css={{width: '20rem'}}
/>
```

---

### Props:

| Name          | Type                         | Default | Description                                       |
| :------------ | :--------------------------- | :------ | :------------------------------------------------ |
| `placeholder` | `string` \| `null`           | `null`  | `Placeholder of the text field`                   |
| `label`       | `string` \| `null`           | `null`  | `Label of the text field`                         |
| `helperText`  | `string` \| `null`           | —       |                                                   |
| `errored`     | `true` \| `false`            | `false` | `Throw error when input value is wrong`           |
| `value`       | `string`                     | —       | `Input value`                                     |
| `rightIcon`   | `ReactNode`                  | -       | `Icon placed on the right side of the text field` |
| `leftIcon`    | `ReactNode`                  | -       | `Icon placed on the left side of the text field`  |
| `isDisabled`  | `false`\| `true`             | `false` | `Disable the text field when true`                |
| `css `        | `CSS`                        | —       |                                                   |

# Text Field

Text Fields let users enter and edit text.

---

### Import:

```js
import {Textfield} from '@comfortdelgro/react-compass'
```

or

```js
import Textfield from '@comfortdelgro/react-compass/textfield'
```

We recommend the second method to import components for the benefits of [tree shaking](https://developer.mozilla.org/en-US/docs/Glossary/Tree_shaking).

### Custom styling:

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

<section className={styles.avatarShowcaseContainer}>
  <Textfield
    value='Wrong Value'
    label='Name'
    placeholder='Enter your name'
    helperText='Use Helper Text for Errors'
    isDisabled
    css={{width: '20rem'}}
  />
</section>

```js
<Textfield
  value='Wrong Value'
  label='Name'
  placeholder='Enter your name'
  helperText='Use Helper Text for Errors'
  isDisabled
  css={{width: '20rem'}}
/>
```

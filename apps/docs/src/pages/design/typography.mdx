# Typography

Good typography can help create clear hierarchies, organize information, and guide users through a product or experience.

---

### Import:

```js
import {Typography} from '@comfortdelgro/react-compass'
```

or

```js
import Typography from '@comfortdelgro/react-compass/typography'
```

### Fonts

- Poppin is the primary digital product typeface of Pax app for both iOS and AOS.
- Native OS System feedback will use Roboto for AOS, and SF Sans Pro for IOS.

### Weights

The choice of font weight is also based on the principles of order, stability, and restraint. In most cases, just regular and bold should be enough.

#### Body

You can use `semibold` and `bold` font weights on these font sizes for semiboldes.

<div className='flex flex-col w-full items-start mt-2'>
  <FontDisplay size={theme.fontSizes.body1} family='Poppins' weight='normal' />
  <FontDisplay size={theme.fontSizes.body2} family='Poppins' weight='normal' />
  <FontDisplay size={theme.fontSizes.body3} family='Poppins' weight='normal' />
</div>

```js
<Typography.Body variant='body1'>The quick brown</Typography.Body>
<Typography.Body variant='body2'>The quick brown</Typography.Body>
<Typography.Body variant='body3'>The quick brown</Typography.Body>
```

#### Labels

You can use `light`, `semibold` and `bold` font weights on these font sizes for semiboldes.

<div className='flex flex-col w-full items-start mt-2'>
  <FontDisplay size={theme.fontSizes.label1} family='Poppins' weight='normal' />
  <FontDisplay size={theme.fontSizes.label2} family='Poppins' weight='normal' />
  <FontDisplay size={theme.fontSizes.label3} family='Poppins' weight='normal' />
</div>

```js
<Typography.Label variant='label1'>The quick brown</Typography.Label>
<Typography.Label variant='label2'>The quick brown</Typography.Label>
<Typography.Label variant='label3'>The quick brown</Typography.Label>
```

### Links

You can use `semibold` font weight on these font sizes for semiboldes.

<div className='flex flex-col w-full items-start mt-2'>
  <FontDisplay size={theme.fontSizes.link1} family='Poppins' weight='semibold' color='#009EDA' />
  <FontDisplay size={theme.fontSizes.link2} family='Poppins' weight='medium' color='#009EDA' />
  <FontDisplay size={theme.fontSizes.link3} family='Poppins' weight='medium' color='#009EDA' />
</div>

```js
<Typography.Link variant='link1'>The quick brown</Typography.Link>
<Typography.Link variant='link2'>The quick brown</Typography.Link>
<Typography.Link variant='link3'>The quick brown</Typography.Link>
```

| Name     | Type             | Default | Description                        |
| :------- | :--------------- | :------ | :--------------------------------- |

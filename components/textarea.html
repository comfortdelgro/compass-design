Textarea

Text areas

Progress indicators commonly known as spinners, express an unspecified wait time or display the length of a process.

💡

Check out our ladle for more info on this component.

Import:

import {Textarea} from '@comfortdelgro/react-compass'

or

import Textarea from '@comfortdelgro/react-compass/textarea'

Basic text areas:

0
0/50
0/50
  <Textarea placeholder='Enter your message' />
  <Textarea label='Text Label' placeholder='Enter your message' isDisabled/>
  <Textarea
    label='Text Label'
    placeholder='Enter your message'
    wordCount
    value='Lorem ipsum dolor sit amet.'
  />
  <Textarea
    label='Text Label'
    placeholder='Enter your message'
    wordCount
    maxLength={50}
    value={'Lorem ipsum dolor sit amet.'}
    onChange={() => {}}
  />
  <Textarea
    label='Text Label'
    placeholder='Enter your message'
    wordCount
    maxLength={50}
    isErrored
  />

Custom styling:

You can style React Compass's components with the css prop.
With css prop, you can basically do anything that css can do.

0/50
<Textarea
  label='Text Label'
  placeholder='Enter your message'
  wordCount
  maxLength={50}
  isErrored
  css={{height: '10em', width: '50em'}}
/>

Props:

NameTypeDefaultDescription
placeholderstring | nullnullPlaceholder of the text field
labelReactNodenullLabel of the text field
isErroredtrue | falsefalseThrow error when input value is wrong
valuestring—Input value
isDisabledfalse| truefalseDisable the text field when true
css CSS—
wordCountfalse| truefalseShow word count of the input when true
maxLengthnumber—Limit length of the input
onChangefunction—

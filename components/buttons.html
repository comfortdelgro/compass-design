Components
Buttons

Button

Buttons allow users to take actions, and make choices, with a single tap.

💡

Check out our ladle for more info on this component.

Import:

import {Button} from '@comfortdelgro/react-compass'

or

import Button from '@comfortdelgro/react-compass/button'

Button Sizes:

<Button size='lg'>Large</Button>
<Button size='md'>Medium (Default)</Button>
<Button size='sm'>Small</Button>

Button with ripple effect:

<Button variant='primary' ripple>
  Primary
</Button>
<Button variant='secondary' ripple>
  Secondary
</Button>
<Button variant='danger' ripple>
  Danger
</Button>
<Button variant='ghost' ripple>
  Ghost
</Button>

Button Variants:

<Button variant='primary'>Primary</Button>
<Button variant='secondary'>Secondary</Button>
<Button variant='danger'>Danger</Button>
<Button variant='ghost'>Ghost</Button>
<Button isDisabled>Disabled</Button>

Buttons with left icons :

<Button variant='secondary' leftIcon={<Icon icon={faThumbsUp} />}>
  Secondary
</Button>
<Button variant='danger' leftIcon={<Icon icon={faThumbsUp} />}>
  Danger
</Button>
<Button variant='ghost' leftIcon={<Icon icon={faThumbsUp} />}>
  Ghost
</Button>

Buttons with right icons :

<Button variant='secondary' rightIcon={<Icon icon={faThumbsUp} />}>
  Secondary
</Button>
<Button variant='danger' rightIcon={<Icon icon={faThumbsUp} />}>
  Danger
</Button>
<Button variant='ghost' rightIcon={<Icon icon={faThumbsUp} />}>
  Ghost
</Button>

Buttons with both left and right icons :

<Button variant='secondary' rightIcon={<Icon icon={faThumbsUp} />} leftIcon={<Icon icon={faVirus} />}>
  Secondary
</Button>
<Button variant='danger' rightIcon={<Icon icon={faThumbsUp} />} leftIcon={<Icon icon={faVirus} />}>
  Danger
</Button>
<Button variant='ghost' rightIcon={<Icon icon={faThumbsUp} />} leftIcon={<Icon icon={faVirus} />}>
  Ghost
</Button>

Buttons with icons as content:

import Icon from '@comfortdelgro/react-compass/icon'
 
<Button variant='secondary'>
  <Icon icon={faVirus} />
</Button>
<Button variant='danger'>
  <Icon icon={faVirus} />
</Button>
<Button variant='ghost'>
  <Icon icon={faVirus} />
</Button>

Loading buttons :

<Button variant='secondary' loading>
  Secondary
</Button>
<Button variant='danger' loading rightIcon={<Icon icon={faThumbsUp} />}>
  Danger
</Button>
<Button variant='ghost' loading rightIcon={<Icon icon={faThumbsUp} />}>
  Ghost
</Button>

Custom styling:

You can style React Compass's components with the css prop.
With css prop, you can basically do anything that css can do.

<Button
  variant='danger'
  rightIcon={<Icon icon={faThumbsUp} />}
  css={{'&:hover': {color: 'grey'}}}
>
  Danger
</Button>

Act as a link:

You can pass href props for button to make it become a link

Navigate

External Navigate

Target Blank Navigate

Navigate

<Button variant='primary' href='/#page'>
    Navigate
  </Button>
  <Button variant='secondary' href='https://google.com' hrefExternal>
    External Navigate
  </Button>
  <Button variant='danger' href='/#page' hrefTarget='_blank'>
     Target Blank Navigate
  </Button>
  <Button variant='ghost' href='/#page'>
    Navigate
  </Button>

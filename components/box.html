Components
Box

Box

The Box component serves as a wrapper component for most of the CSS utility needs.

💡

You can see how this component is implemented on ladle (opens in a new tab)

Import:

import {Box} from '@comfortdelgro/react-compass'

or

import Box from '@comfortdelgro/react-compass/box'

The 'as' prop:

The 'as' prop refers to the HTML element that the Box component should render as. By default, the Box component will render as a div element, but if you pass a different value to the as prop, it will render as that element instead.

My 'div' box

My 'h2' box

<Box
  css={{
    backgroundColor: 'grey',
    width: '8rem',
    textAlign: 'center',
    height: '2rem',
    textAlign: 'center',
    paddingTop: '0.3rem',
  }}
>
  <p>My 'div' box</p>
</Box>
 
<Box as='h2' color='red'>
    <p>My 'h2' box</p>
</Box>

Dynamic Styling:

You can use prop 'css' for styling. This is so much powerful than the regular 'style' prop.

My box

<Box
  css={{
    backgroundColor: 'grey',
    width: '8rem',
    textAlign: 'center',
    '& p': {
      color: 'white',
    },
  }}
>
  <p>My box</p>
</Box>

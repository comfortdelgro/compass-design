Text Field

Text Fields let users enter and edit text.

Import:

import {Textfield} from '@comfortdelgro/react-compass'

or

import Textfield from '@comfortdelgro/react-compass/textfield'

Basic text field:

<Textfield placeholder='Enter your name' />

Text field with label:

<Textfield label='Name' placeholder='Enter your name' />

Text field with helper text:

Helper Text
<Textfield
  label='Name'
  placeholder='Enter your name'
  helperText='Helper Text'
/>

Text field with input error:

Use Helper Text for Errors
<Textfield
  errored
  value='Wrong Value'
  label='Name'
  placeholder='Enter your name'
  helperText='Use Helper Text for Errors'
/>

Text field with icons:

<Textfield label='Name' leftIcon={faThumbsUp} placeholder='Enter your name' />
<Textfield
  label='Name'
  rightIcon={faThumbsUp}
  placeholder='Enter your name'
/>

Disabled text field:

Use Helper Text for Errors
<Textfield
  value='Wrong Value'
  label='Name'
  placeholder='Enter your name'
  helperText='Use Helper Text for Errors'
  isDisabled
/>

Custom styling:

You can style React Compass's components with the css prop.
With css prop, you can basically do anything that css can do.

Use Helper Text for Errors
<Textfield
  value='Wrong Value'
  label='Name'
  placeholder='Enter your name'
  helperText='Use Helper Text for Errors'
  isDisabled
  css={{width: '20rem'}}
/>

Props:

NameTypeDefaultDescription
placeholderstring | nullnullPlaceholder of the text field
labelstring | nullnullLabel of the text field
helperTextstring | null—
erroredtrue | falsefalseThrow error when input value is wrong
valuestring—Input value
rightIconIconProp|false| truefalseIcon placed on the right side of the text field
leftIconIconProp|false| truefalseIcon placed on the left side of the text field
isDisabledfalse| truefalseDisable the text field when true
css CSS—

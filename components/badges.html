Badge

Badges can be used to highlight important bits of information such as labels, notifications, data trends & status.

Import:

import {Badge} from '@comfortdelgro/react-compass'

or

import Badge from '@comfortdelgro/react-compass/badge'

Basic:

Default
<Badge label='Default' />

Colors :

InfoSuccessDangerWarning
<Badge label='Info' color='info' />
 
<Badge label='Success' color='success' />
 
<Badge label='Danger' color='danger' />
 
<Badge label='Warning' color='warning />

Variants:

PrimarySecondaryOutline
<Badge label='Primary' color='danger' variant='primary' />
<Badge label='Secondary' color='danger' variant='secondary' />
<Badge label='Outline' color='danger' variant='outline' />

Default icon:

PrimarySuccessSecondaryOutline
<Badge label='Primary' color='info' variant='primary' icon />
<Badge label='Success' color='success' variant='primary' icon />
<Badge label='Secondary' color='danger' variant='secondary' icon />
<Badge label='Outline' color='warning' variant='outline' icon />

Custom icon:

LGTMPractice Social Distancing
<Badge
  label='LGTM'
  color='success'
  variant='secondary'
  icon={faThumbsUp}
/>
<Badge
  label='Practice Social Distancing'
  color='danger'
  variant='outline'
  icon={faVirus}
/>

Custom styling:

You can style React Compass's components with the css prop.
With css prop, you can basically do anything that css can do.

Outline
<Badge
  label='Outline'
  color='warning'
  variant='outline'
  css={{color: 'purple', '&:hover': {backgroundColor: 'grey'}}}
/>

Props:

NameTypeDefaultDescription
colorinfo | success | warning | dangerinfoThese colors will determine the color of background of the badges.
labelString —Lable of the badges.
variantprimary | secondary | outlineprimaryVariants will determine the color of the badges' label, border and the darkness of its background .
iconIconProp|false| truefalseWhen false, no icon is shown in the badge. When true, the default icons are shown. And you can pass in custom icons to this prop.
cssCSS—The system prop that allows defining system overrides as well as additional CSS styles.

Components
Sidenav

Sidenav

The Sidenav displays horizontally information and navigation relating to the current screen.

Import

import {Sidenav} from '@comfortdelgro/react-compass'

or

import Sidenav from '@comfortdelgro/react-compass/Sidenav'

Basic Sidenav

M
<Sidenav css={{height: '800px'}}>
  <Sidenav.Item>
    <Avatar label='M' />
    <h2 style={{fontSize: '24px', marginLeft: '22px', color: '#323130'}}>
      Workbench
    </h2>
  </Sidenav.Item>
  <Sidenav.Divider />
  <Sidenav.Item>
    <Icon icon={faAmbulance}></Icon>
    <span>Menu</span>
  </Sidenav.Item>
  <Sidenav.Item>
    <Icon icon={faBank}></Icon>
    <span>Menu</span>
  </Sidenav.Item>
  <Sidenav.Item>
    <Icon icon={faCar}></Icon>
    <span>Menu</span>
  </Sidenav.Item>
  <Sidenav.Item>
    <Icon icon={faHelicopter}></Icon>
    <span>Menu</span>
  </Sidenav.Item>
</Sidenav>

Expand

You can expand the Sidenav without hovering

Menu
<Sidenav css={{height: '800px'}} expand>
  <Sidenav.Item isActive>
    <Icon icon={faHelicopter}></Icon>
    <span>Menu</span>
  </Sidenav.Item>
</Sidenav>

Sidenav Item

Sidenav Item accepts two children. If Sidenav is not in expand mode, only the first child will be rendered

<Sidenav css={{height: '800px'}}>
  <Sidenav.Item isActive>
    <Icon icon={faHelicopter}></Icon>
    <span>Menu</span>
  </Sidenav.Item>
</Sidenav>

Sidenav Divider

Sidenav divider provides a horizontal divider to Sidenav

<Sidenav css={{height: '800px'}}>
  <Sidenav.Item isActive>
    <Icon icon={faHelicopter}></Icon>
    <span>Menu</span>
  </Sidenav.Item>
  <Sidenav.Divider />
  <Sidenav.Item>
    <Icon icon={faHelicopter}></Icon>
    <span>Menu</span>
  </Sidenav.Item>
</Sidenav>
